Hola, Bienvenido!

Este proyecto permite reconocer una sintaxis basica del lenguaje TypeScript y construir un arbol sintactico. El cual se recorre, ejecuta y analiza cada una las acciones ingresadas por el usuario.

Utiliza una traduccion orientada a la sintaxis para analizar sentencias basicas del lenguaje TypeScript, tales como expresiones aritmeticas, logicas y relacionales, asi como tambien instrucciones de declaracion, asignacion,ciclos y funciones.



Ir a codigo




AST

Segun el codigo que se ingrese el usuario, se genera un arbol sintactico.

Errores

Se presenta una tabla la cual contiene a detalle de los errores lexicos, sintacticos y semanticos que se reconocieron, ademas una descripcion del error, la linea y columna en donde se encontro en el archivo de entrada

Tabla de simbolos

Se presenta una tabla de simbolos que contiene detalle de las variables, arreglos y funciones que se reconocen en un entorno. Datos como nombre,tipo y valor.
